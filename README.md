# GITPUSH
- auto run git add, commit and push
- 自动允许git命令（add,commit,push)
- The file ```.gitignore``` must exists
- 文件```.gitignore```必须存在
- git remote all must exists
- 远程仓库 all 必须存在
- git remote add all <>
- git remote set-url --add all <>

# releases 说明
- releases v1.0 附带了基础```.gitignore```文件，使用前将文件重命名```default```去除即可
